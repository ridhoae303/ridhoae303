## 💫 About Me

I'm **ridhoae303**, a beginner developer and modder who mostly builds stuff from an Android phone.

My setup is not fancy at all. I don't have a laptop, PC, or desktop. Going to an internet café? Yeah... I don't have the money for that either. So most of my projects are made with whatever I can do from my phone.

I like messing around with reverse engineering, modding, Android internals, anti-tamper research, and random experiments that probably should not work... but somehow do.

Sometimes my projects look rough, sometimes my README looks messy, and sometimes the layout is not perfect. That is just the Android-only grind. I build anyway.

> If my banner looks bad, please ignore it. I made this whole thing from my phone, so yeah... it is what it is.
